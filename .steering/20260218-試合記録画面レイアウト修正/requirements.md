# 要求: 試合記録画面レイアウト修正 (issue#1)

## 問題点

1. **情報が散らばって見づらい** — ヘッダー・スコアシート・入力パネルが縦積みで整理されていない
2. **縦スクロールが二重に発生** — スコアシートの max-h 制限と入力パネルの flex-1 overflow-auto の2箇所でスクロールが発生
3. **試合中に不要なリンクが表示** — 「成績」「印刷」ボタンが試合中のヘッダーに表示される
4. **投球記録がセルに表示されない** — スコアシートのセルに一球ごとの投球記録が表示されていない

## 要求事項

1. h-screen 固定・縦スクロール完全禁止のレイアウトに変更
2. 左ペイン: スコアシート（横スクロールのみ）
3. 右ペイン: 試合状況パネル + 操作UI（右ペイン内スクロール）
4. 「成績」「印刷」リンクをゲーム記録中画面から削除
5. スコアシートのセルに一球ごとの投球記号を表示
6. スコアボード（得点表示）を右上パネルに追加
