# 要件: スコアシートの攻守切り替え表示

## 概要

試合記録画面のスコアシートエリアで、表チームと裏チームを切り替えて表示できるようにする。
現在は攻撃中チームのみ表示されており、相手チームの打席記録を確認する手段がない。

## 要求仕様

- スコアシート上部にトグルボタン（チーム名 + 「表」「裏」）を配置
- ボタンで表チーム / 裏チームを切り替えて対応するスコアシートを表示
- デフォルトは現在攻撃中のチームを選択
- 攻守交代時（イニング変更時）はデフォルトに自動リセット（攻撃中チームに戻る）
- 非攻撃チームを表示中は現在打者ハイライトを表示しない（ScoreSheet の既存ロジックで自然に対応）

## 影響範囲

- `components/scorebook/ScoreSheet.tsx`: Props 変更不要（attackingSide の意味合いが viewingSide に変わる）
- `components/score-input/ScoreInputPage.tsx`: viewingSide state + トグル UI + lineup 計算の追加
