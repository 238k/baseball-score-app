# 要件: 試合途中終了機能

## 概要

降雨中止・コールドゲーム・日没などにより試合が途中で終わった場合に対応するため、試合記録画面から任意のタイミングで試合を終了し、その時点のスコアを確定できるようにする。

## 受入条件

- [ ] 試合記録画面に「試合を終了する」ボタンがある
- [ ] 確認ダイアログで誤操作を防止している（終了理由の選択付き）
- [ ] 終了後は投球・結果入力ができない（読み取り専用）
- [ ] 終了後の画面に成績・印刷リンクが表示される
- [ ] 試合一覧で試合のステータス（進行中 / 終了 / 途中終了）が確認できる

## 試合ステータス

```ts
type GameStatus = 'in_progress' | 'completed' | 'suspended';
```

- `in_progress`: 試合進行中（デフォルト）
- `completed`: 正規終了（9回または延長終了）
- `suspended`: 途中終了（降雨中止・コールドゲームなど）

## 終了理由の選択肢

- コールドゲーム
- 降雨中止
- 日没
- その他

## UI 仕様

- 右ペイン下部に「試合を終了する」ボタンを常時表示（in_progress 時のみアクティブ）
- タップ時にオーバーレイ確認ダイアログを表示（終了理由の選択、確定・キャンセルボタン）
- 終了後: 入力パネルをロックして読み取り専用メッセージを表示
- 終了後: 成績・印刷リンクを右ペインに表示
